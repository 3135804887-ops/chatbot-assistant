# 快速部署指南 - 5分钟完成

## 第一步：初始化Git仓库（1分钟）

在项目目录 `d:\BJbot` 打开终端，执行：

```bash
git init
git add .
git commit -m "Initial commit: 智能对话助手"
```

## 第二步：创建GitHub仓库（1分钟）

1. 访问 https://github.com/new
2. 填写信息：
   - Repository name: `chatbot-assistant`
   - Description: `基于智谱AI的智能对话助手`
   - 选择 Public
   - 点击 "Create repository"

## 第三步：推送到GitHub（1分钟）

复制GitHub页面上的命令，或执行：

```bash
git remote add origin https://github.com/YOUR_USERNAME/chatbot-assistant.git
git branch -M main
git push -u origin main
```

**注意：** 将 `YOUR_USERNAME` 替换为你的GitHub用户名

## 第四步：部署到Streamlit Cloud（2分钟）

1. 访问 https://share.streamlit.io/
2. 点击 "Sign up" 使用GitHub账号登录
3. 点击 "New app"
4. 填写信息：
   - Repository: 选择你的仓库
   - Branch: `main`
   - Main file path: `streamlit前端.py`
5. 点击 "Deploy"

## 第五步：配置API密钥（30秒）

1. 在应用页面点击 "Settings"
2. 点击 "Secrets"
3. 添加：

```toml
ZHIPUAI_API_KEY = "t3110960985"
```

4. 点击 "Save"

## 完成！

你会获得一个URL：
```
https://chatbot-assistant-xxxxx.streamlit.app
```

## 常见问题

**Q: 推送到GitHub失败？**
```bash
# 先登录GitHub
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```

**Q: Streamlit Cloud部署失败？**
- 检查主文件路径是否正确：`streamlit前端.py`
- 查看部署日志排查错误

**Q: 如何更新应用？**
```bash
git add .
git commit -m "更新说明"
git push
```

Streamlit Cloud会自动重新部署！
