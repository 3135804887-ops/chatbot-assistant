# 智能比价机器人实现方案

## 项目结构
```
d:\BJbot\
├── streamlit前端.py (主文件，已存在)
├── config.py (配置文件 - 新建)
├── llm_service.py (智谱AI大模型服务 - 新建)
├── price_comparison.py (比价服务 - 新建)
└── requirements.txt (依赖包 - 新建)
```

## 实现步骤

### 1. 创建配置文件 `config.py`
- 存储智谱AI API密钥
- 存储第三方比价API配置
- 定义系统参数（如超时时间、重试次数等）

### 2. 创建大模型服务模块 `llm_service.py`
- 封装智谱AI API调用（使用zhipuai SDK）
- 实现对话接口，支持多轮对话
- 提供意图识别功能（判断用户是否在比价）
- 提取商品关键词和查询条件
- 完善的错误处理和重试机制

### 3. 创建比价服务模块 `price_comparison.py`
- 对接第三方比价API（如比价网、慢慢买等）
- 实现商品搜索功能
- 实现价格对比和排序功能
- 格式化比价结果（表格、链接等）
- 添加缓存机制避免重复请求

### 4. 重构 `streamlit前端.py`
- 集成llm_service和price_comparison模块
- 替换模拟数据为真实API调用
- 优化用户交互流程
- 添加加载状态和错误提示

### 5. 创建 `requirements.txt`
- streamlit
- zhipuai (智谱AI SDK)
- requests (HTTP请求)
- 其他必要依赖

## 核心功能流程
1. 用户输入商品查询 → 2. 大模型理解意图并提取关键词 → 3. 调用比价API获取数据 → 4. 大模型生成智能回复 → 5. 展示比价结果

## 代码规范
- 所有模块化设计，职责清晰
- 每个函数都有详细的中文注释
- 使用类型提示提高可读性
- 完善的错误处理和日志记录
